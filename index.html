# OBI Inventory System – Description

This is a private tire inventory management web application built using HTML, CSS, and JavaScript with all data stored locally in the browser using LocalStorage.

The system is designed for tire shop employees to manage inventory, sales, shipments, and employee accounts through a clean black-and-white dashboard interface.

Main Features

  • 🔐 User Authentication
    • Login and Sign Up system
    • First account automatically becomes Owner
    • Additional accounts become Workers
    • Role-based permissions:
      • Owner
      • Manager
      • Worker
  • 📊 Dashboard
    • Displays:
      • Total stock quantity
      • Total SKUs
      • Units sold
      • Revenue
      • Profit
    • Shows low-stock and out-of-stock alerts
    • Displays recent activity logs
  • 🚗 Inventory Management
    • Add new tires
    • Edit tire information
    • Delete tires
    • Track:
      • SKU
      • Brand
      • Size
      • Location
      • Quantity
      • Cost
      • Selling price
    • Search inventory instantly
    • Filter low-stock and out-of-stock items
    • Export inventory to CSV files
  • 💰 Sales Tracking
    • Record tire sales
    • Automatically reduces inventory quantity
    • Calculates:
      • Sale total
      • Profit
    • Stores complete sales history
  • 📦 Shipment Management
    • Receive incoming tire shipments
    • Increase stock automatically
    • Add notes for deliveries
    • Maintain shipment history
  • 📜 Activity Logs
    • Records every important action:
      • Tire added
      • Tire edited
      • Tire deleted
      • Sales completed
      • Shipments received
    • Tracks:
      • User
      • Date
      • SKU
      • Action details
  • 👥 Employee Management
    • Owner-only section
    • Create employee accounts
    • Edit employee information
    • Change user roles
    • Reset passwords

Technical Details

Frontend

  • Pure HTML
  • Pure CSS
  • Pure JavaScript
  • No frameworks used

Storage

  • Browser LocalStorage
  • No database required
  • No server required

Security Features

  • User sessions
  • Input sanitization against HTML injection
  • Role-based access control

Overall Purpose

The OBI Inventory System is a lightweight, private inventory platform built specifically for a tire business. It allows employees to manage stock, process sales, receive shipments, track activity, and manage staff without requiring internet access, servers, or external databases.

Project Type: Inventory Management System (IMS)

Industry: Tire & Automotive Business

Technology Stack: HTML, CSS, JavaScript, LocalStorage

Deployment: Runs directly in a web browser as a standalone application.
